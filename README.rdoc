== README

As of 8/3/2015, this application, "this-week-in", is in a very baseline stage, and displays a simple, unstyled summary of feature articles from an issue of National Geographic Magazine.

The application will be built out to ask the user to choose from a selection of periodic publications, and to specify a specific issue (mm/yyyy or mm/dd/yyyy) of the publication.  Upon entry of the publication and date, the user will see a summary of feature articles from that issue.  Styling is also on tap.
